Frameworks & Libraries Used

The project intentionally avoids large frameworks to maintain simplicity and performance. It uses:

Vanilla HTML, CSS, and JavaScript for all core behavior

Google Fonts CDN for typography

JSDelivr for external asset hosting



Performance Metrics

Minimized HTTP requests by using compressed media and modular CSS

If implemented caching and global asset delivery via JSDelivr

Deferred JavaScript loading to prevent render-blocking

Clean folder structure ensuring predictable routing and quick server lookups

Accessibility & SEO Strategies

To ensure usability and search ranking quality, the site incorporates:

Accessibility

Semantic HTML (header, main, nav, footer)

ARIA labels and roles on navigation and modal elements

Keyboard-friendly navigation (Tab, Enter, Esc, Arrow keys)

High-contrast color scheme and readable font sizes

Descriptive alt text for all images


SEO

Proper <title> and <meta description> for all pages

Clean URL structure and descriptive file names

Optimized images with proper alt text

Mobile-responsive design with correct viewport scaling

Fast loading times due to CDN hosting and optimized assets

Logical heading hierarchy (h1, h2, h3) for search engines
